# 📊 One Billion Row Challenge (1BRC)

## 🎯 About the project

This repository is a based in the original One Billion Row Challenge, a Go project designed to aggregate one billion lines of text data.

---

## 🛠️ Stack

- **Go 1.26.1**

---

## 🚀 installation and Configuration

### Clone the repository

```bash
git clone https://github.com/andersonfagundes/1brc.git
cd 1brc
```

---

## 🎮 How to run

```bash
go run m.go
```
